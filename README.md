# Psychoactive Cafe

Traditional Chinese first-pass website for Psychoactive Cafe.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS v4

## Routes

- `/`
- `/story`
- `/ritual`
- `/drops/ancestor-roast-no-1`
- `/reserve`
- `/reserve/success`

## Commands

```bash
pnpm install
pnpm dev
pnpm build
```
